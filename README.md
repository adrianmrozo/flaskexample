# flask-example
## first steps
after downloading this folder you have to make both shell scrips (*.sh) executable with following command: 
``` bash
chmod +x file_name.sh
```

so:

``` 
chmod +x build_image.sh
chmod +x run_container.sh
``` 


### build the image (first usage)
``` bash 
./build_image.sh
```

### run the container
``` bash
./run_container.sh
```
